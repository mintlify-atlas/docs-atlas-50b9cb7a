### How to Download

<Steps>

  </Step>

## Download Quality Settings

### Quality Options

Choose the download quality that balances audio fidelity and storage space:

<Accordion title="High Quality">
  - **Best audio fidelity**: Maximum bitrate for audiophile-quality sound
  - **Larger file sizes**: 5-10 MB per track average
  - **Recommended for**: WiFi downloads, ample storage space
  - **Use when**: You want the absolute best sound quality
</Accordion>

<Accordion title="Medium Quality">
  - **Balanced quality**: Good audio quality with reasonable file sizes
  - **Moderate file sizes**: 3-5 MB per track average
  - **Recommended for**: Most users, balanced storage/quality needs
  - **Use when**: You want good quality without using too much space
</Accordion>

<Accordion title="Low Quality">
  - **Smaller files**: Minimal storage usage
  - **Lower bitrate**: 1-3 MB per track average
  - **Recommended for**: Limited storage, background listening
  - **Use when**: Storage space is at a premium
</Accordion>

<Note>
  You can set a default download quality in Settings > Downloads. This will be used for all future downloads unless you manually select a different quality.
</Note>

### Tracking Downloads

Monitor your download activity:

- **Real-time Progress**: See percentage completed for each download
- **Queue Status**: View all pending and active downloads
- **Download Speed**: Monitor download speed and estimated time
- **Notification Updates**: Get notified when downloads complete

### Managing Active Downloads

- **Pause Downloads**: Temporarily pause downloads to conserve bandwidth
- **Resume Downloads**: Continue paused downloads later
- **Cancel Downloads**: Stop and remove unwanted downloads
- **Prioritize Downloads**: Reorder download queue (coming soon)

<Info>
  Downloads continue in the background even when you switch apps or lock your screen. You'll receive a notification when all downloads are complete.
</Info>

## Offline Mode

### Full Offline Functionality

When you're offline, Musika provides:

- **Complete Playback**: Play all downloaded tracks with full controls
- **Offline Library**: Browse your downloaded music collection
- **Playlist Access**: Access and play downloaded playlists
- **Search Downloaded**: Search within your offline library
- **Queue Management**: Create and modify playback queues

### Offline Search

Find music in your offline library:

- **Fast Search**: Instant results from downloaded content
- **Filter Options**: Filter by songs, albums, artists, or playlists
- **Metadata Search**: Search by title, artist, album, or genre
- **Recently Downloaded**: Quick access to newest downloads

<Tip>
  Musika automatically switches to offline mode when it detects no internet connection. You'll see an indicator showing you're in offline mode.
</Tip>

### Storage Cleanup

<Steps>
  <Step title="Open Settings">
    Navigate to Settings > Storage
  </Step>
  
  <Step title="Review Usage">
    See breakdown of storage usage by category
  </Step>
  
  <Step title="Clear Cache">

## Download Settings

### WiFi-Only Downloads

Protect your cellular data:

- **Enable WiFi-Only**: Downloads only start on WiFi networks
- **Pause on Cellular**: Automatically pause downloads when switching to cellular
- **Smart Downloads**: Download new playlist additions only on WiFi

<Tip>
  Enable "WiFi-Only Downloads" in Settings > Downloads to prevent accidental cellular data usage.
</Tip>

### Automatic Downloads

Configure automatic download behavior:

<Accordion title="Playlist Auto-Download">
  - Automatically download new songs added to selected playlists
  - Keep playlists up-to-date for offline access
  - Configure per-playlist or globally
  - Respects WiFi-only settings
</Accordion>

<Accordion title="Smart Download Management">
  - Remove old downloads to make room for new ones
  - Keep most played songs, remove rarely played
  - Set maximum storage limit for downloads
  - Customize retention period for downloads
</Accordion>

## Syncing Downloads

### Cross-Device Sync

Keep downloads synchronized across devices:

- **Download Preferences**: Sync which content should be available offline
- **Playlist Sync**: Automatically download playlist changes
- **Selective Sync**: Choose which playlists sync automatically
- **Manual Override**: Manually download/remove content on specific devices

<Note>
  Sync settings help you maintain consistent offline libraries across devices, but actual audio files are not transferred between devices - each device downloads from YouTube Music independently.
</Note>

## Troubleshooting

<Accordion title="Download Failures">
  **Downloads Won't Start**:
  - Check internet connection
  - Verify sufficient storage space (need at least 500MB free)
  - Ensure WiFi-only setting isn't blocking cellular downloads
  - Check if app has storage permissions

  **Downloads Keep Failing**:
  - Restart the app
  - Clear app cache
  - Check if specific tracks are region-restricted
  - Update to the latest version
</Accordion>

<Accordion title="Playback Issues">
  **Downloaded Songs Won't Play**:
  - Verify files weren't corrupted
  - Check if storage location is accessible
  - Re-download affected tracks
  - Ensure downloads completed successfully

  **Missing Downloads**:
  - Check if auto-cleanup removed them
  - Verify storage location hasn't changed
  - Look in "Recently Deleted" (if available)
  - Check if SD card was removed
</Accordion>

<Accordion title="Storage Problems">
  **Running Out of Space**:
  - Clear cache to free temporary files
  - Remove unnecessary downloads
  - Lower download quality for new downloads
  - Move downloads to SD card (if supported)

  **Can't Free Up Space**:
  - Check Settings > Storage for detailed breakdown
  - Clear app data (removes all downloads)
  - Manually delete specific albums or playlists
  - Check for duplicates or multiple quality versions
</Accordion>
